# Hockey Lineup Builder

**Version:** 2.0.0

A generic hockey lineup-building and game-strategy skill for any team. On first use, the skill walks the coach through a short setup interview (team name, level, head coach, branding, roster source) and persists the answers to `.claude/hockey-lineup-builder.local.md`. Subsequent lineup requests skip the setup.

## Overview

Build game-day lineups, plan strategies against opponents, optimize special teams, manage player absences, and plan tournaments — driven by your team's own roster and playing identity.

## Quick Start

On first use, the skill will run a setup interview. After that:

- "Build a lineup for Saturday's game — Marsh and Freeman are available but Bagga is out"
- "Plan strategy against Paul's team"
- "Set up special teams units for the tournament"
- "We're missing 3 forwards this weekend, build contingency lines"
- "Update player ratings with this new spreadsheet"

## Importing a Roster

```bash
# From Excel
python3 scripts/update_roster.py path/to/roster.xlsx

# From CSV
python3 scripts/update_roster.py path/to/roster.csv

# Inline JSON
python3 scripts/update_roster.py --json '{"players": [...]}'

# Preview without writing
python3 scripts/update_roster.py --dry-run path/to/roster.xlsx
```

See [references/roster-management.md](references/roster-management.md) for column aliases and examples.

## Configuration

The skill stores team-specific configuration at `.claude/hockey-lineup-builder.local.md` in the user's working directory:

- Team name, age group, level, head coach
- Playing-style description and identity tagline
- Branding source (Claude branding skill name, or free-form notes, or none)
- Roster source

The setup interview writes this file on first run. Edit it directly if anything changes.

## Branding

When generating Word documents (Bench Card, Lineup Poster, Strategy Summary), the skill applies branding based on the config:

- **Branding skill**: invokes a named Claude skill (e.g. `anthropic-skills:jr-kraken-brand`) for the team's color palette, typography, and logos
- **Branding notes**: applies free-form notes (colors, fonts, taglines) manually
- **None**: produces clean, unbranded documents

## Worked Example

[`examples/jr-kraken-18u-navy/`](examples/jr-kraken-18u-navy/) contains a fully populated example: 17-player roster with ratings, complete "Balanced Depth" strategy doc, and a sample lineup request. Use it as a reference when filling in your own team's files.

## Key Features

- Setup interview captures team identity, branding, and roster source on first use
- Roster import from Excel/CSV/JSON with flexible column matching
- Default 1.00–3.00 rating scale; also accepts 1–5 if needed
- Tier-based, name-free strategy and tactical files (resilient to roster changes)
- Optional TeamSnap integration for RSVP checking
- Branded Word document output (via Word skill + configured branding skill)
- Lineup generator script with text-output templates
