# Jr. Kraken 18U Navy Lineup Builder

**Version:** 1.0.0

A specialized lineup building and game strategy skill for Coach Mark's Jr. Kraken 18U C Navy hockey team.

## Overview

Build game-day lineups, plan strategies against opponents, optimize special teams, manage player absences, and plan tournaments -- all rooted in the team's "Balanced Depth" identity.

## Quick Start

- "Build a lineup for Saturday's game -- Marsh and Freeman are available but Bagga is out"
- "Plan strategy against Paul's team"
- "Set up special teams units for the tournament"
- "We're missing 3 forwards this weekend, build contingency lines"

## Key Features

- **17-player roster** with skill ratings (1.00-3.00 scale)
- **9 defensive-capable players** -- biggest strategic advantage
- **6 F/D versatile players** for maximum flexibility
- **Development player sheltering** guidelines for Cram and Thompson
- **Tournament planning** with pool play vs playoff strategies
- **Lineup generator script** with pre-built templates
