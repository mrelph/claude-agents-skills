# Formations & Tactical Systems - 18U C Navy Team

## Overview

This document outlines tactical formations, line deployment strategies, and systematic approaches for Coach Mark's Navy team at the 18U C competitive level.

---

## Forward Line Structures

### Four-Line System (PRIMARY APPROACH ⭐)

**When to Use**:
- Most regular season games
- Pool play in tournaments
- Games where we want to wear opponents down
- When we have full roster available
- Against teams that use 2-3 lines (exploit depth advantage)

**Philosophy**: Roll four lines, keep everyone fresh, dominate third period

**Line Structure**:

```
LINE 1 - Elite/Offensive (35-40% ice time)
LW: 1.00 F/D    C: 1.00 F/D    RW: 1.50 F/D
Focus: Offensive production, top matchup line, all special teams

LINE 2 - Skilled/Two-Way (30-35% ice time)
LW: 1.50 F    C: 1.50 F/D    RW: 1.75 F
Focus: Two-way play, reliable in all situations, PP2 and PK

LINE 3 - Grind/Energy (20-25% ice time)
LW: 2.00 F    C: 2.00 F    RW: 2.00 F
Focus: Energy, physicality, forecheck, defensive responsibility

LINE 4 - Depth/Development (10-15% ice time)
LW: 2.00 F/D    C: 2.50 dev F    RW: 2.00 F/D
Focus: Sheltered minutes for development F, energy shifts, simple assignments
```

**Rotation Strategy**:
- Line 1 → Line 2 → Line 3 → Line 4 → repeat
- Shorten Line 4 shifts (45-60 seconds vs 60-90 for Lines 1-3)
- Double-shift Line 1 in critical situations
- Line 3 can play more if Line 4 is struggling

**Advantages**:
- Everyone stays engaged
- Fresh legs all game
- Third period dominance (opponents tire, we don't)
- Can handle absences easily
- Development opportunities for development forward

### Three-Line System (COMPETITIVE GAMES)

**When to Use**:
- Close playoff games
- Against very strong opponents
- When development D is unavailable (need F/D player on defense more)
- Late in close games
- When we need maximum firepower

**Philosophy**: Best players get more ice time, but everyone still plays

**Line Structure**:

```
LINE 1 - Elite (40-45% ice time)
LW: 1.00 F/D    C: 1.00 F/D    RW: 1.50 F/D
Focus: Carry the offense, dominate shifts

LINE 2 - Skilled (35-40% ice time)
LW: 1.50 F    C: 1.50 F/D    RW: 1.75 F
Focus: Two-way dominance, eat minutes

LINE 3 - Energy/Depth (20-25% ice time)
LW: 2.00 F    C: 2.00 F    RW: 2.00 F
Focus: High-energy shifts, simple play, tire opponents

SPOT DUTY:
- 2.00 F/D player: 5-8 minutes, energy shifts
- 2.50 dev F: 3-5 minutes, very sheltered, offensive zone only
```

**Rotation Strategy**:
- Line 1 → Line 2 → Line 1 → Line 3 → repeat (double-shift top line)
- Spot duty players fill in on changes
- Line 1 and 2 can play back-to-back shifts if needed
- Line 3 matches against opponent's weak line

**Advantages**:
- Maximum ice time for best players
- Can match up against elite opponents
- Still have depth when needed
- Development and depth F/D players still contribute

## Defense Pair Structures

### Three-Pair System (STANDARD ⭐)

**When to Use**: Most games with full roster

**Pair Structure**:

```
PAIR 1 - Top/Shutdown (40-45% ice time)
LD: 1.75 dedicated D    RD: 1.00 elite F/D or 2.00 F/D
Focus: Against opponent's top line, defensive responsibility, PK

PAIR 2 - Two-Way/Balanced (35-40% ice time)
LD: 2.00 dedicated D    RD: 1.50 F/D or 2.00 F/D
Focus: All situations, reliable, can jump into rush

PAIR 3 - Development/Sheltered (20-25% ice time)
LD: 3.00 dev D    RD: 1.50 F/D or 2.00 F/D (strong partner always)
Focus: Sheltered minutes for development D, offensive zone starts
```

**Note**: Actual pairs can vary based on who's playing forward vs defense

**Rotation Strategy**:
- Pair 1 → Pair 2 → Pair 3 → repeat
- Shorten Pair 3 shifts (45-60 seconds)
- Can run Pairs 1 and 2 back-to-back if needed
- Development D ALWAYS with experienced partner

**Advantages**:
- Fresh legs on defense all game (9 D-capable players)
- Can shelter development D
- Flexibility to move players forward if needed
- Strong PK depth

### Four-Pair System (TOURNAMENT DEPTH)

**When to Use**: 
- Multi-game weekends
- Pool play in tournaments
- When everyone is available
- To maximize ice time distribution

**Pair Structure**:

```
PAIR 1: 1.75 dedicated D - 1.00 elite F/D
PAIR 2: 2.00 dedicated D - 1.50 F/D
PAIR 3: 2.00 F/D - 2.00 F/D
PAIR 4: 3.00 dev D - 1.50 F/D (strong partner always)
```

**Rotation**: Equal-ish shifts (Pair 4 slightly less)

**Advantages**:
- Maximum freshness
- Everyone contributes
- Can handle multiple games
- Development opportunities

### Two-Pair System (EMERGENCY/COMPETITIVE)

**When to Use**:
- Multiple D absences
- Very close, critical games
- Overtime situations
- Late in close games

**Pair Structure**:

```
PAIR 1: 1.75 dedicated D - 1.00 elite F/D or 1.50 F/D
PAIR 2: 2.00 dedicated D - 2.00 F/D
```

**Rotation**: Back-and-forth, 60-90 second shifts

## Special Teams Formations

### Power Play Systems

#### PP1 - Umbrella Formation (PRIMARY ⭐)

**When to Use**: Standard power play, against box or diamond PK

**Formation**:
```
         1.50 F (net-front)
              |
              |
    1.00 F/D --- 1.00 F/D --- 1.50 F/D
    (half-wall)  (QB/slot)    (half-wall)

         1.75 D (point)
```

**Roles**:
- **Center (1.00 F/D)**: Quarterback, setup man, high slot
- **Left (1.00 F/D)**: Left half-wall, shot threat, pass option
- **Right (1.50 F/D)**: Right half-wall, shoot/pass
- **High F (1.50 F)**: Net-front, screens, tips, rebounds
- **Point (1.75 D)**: Blue line, one-timer threat, pass option

**Keys**:
- Puck movement, tire PK
- Get net-front F in front for screens
- Shoot from point (D) for tips/rebounds
- Half-wall F/D players: shoot or pass threats

#### PP2 - Overload Formation (SECOND UNIT)

**When to Use**: Second power play unit, change of pace

**Formation**:
```
         1.75 F (net-front)
              |
              |
    1.50 F/D - 2.00 F - 2.00 F
    (half-wall) (slot)  (half-wall)

         2.00 F/D (point)
```

**Philosophy**: Shoot first, crash net, get pucks on goal

**Keys**:
- Simple, direct
- Shots from everywhere
- Crash for rebounds
- High energy

### Penalty Kill Systems

#### PK System - Box+1 Aggressive (PRIMARY ⭐)

**When to Use**: Most penalty kill situations

**Formation (5v4)**:
```
        Forward (F1)
            ↑
            | (pressure)
            
    F2 ---- D1 ---- D2
            |
           F3/F4
```

**Standard PK1 Personnel**:
- **F1** (Pressure): 1.50 F/D two-way forward
- **F2** (Box Forward): 2.00 F/D or 1.75 F
- **D1** (Strong Side D): 1.75 dedicated D
- **D2** (Weak Side D): 2.00 dedicated D

**Keys**:
- F1 pressures puck carrier aggressively
- Box stays compact (15 feet)
- Rotate when F1 tired
- Block passing lanes
- Clear pucks hard when possible

**Philosophy**: 
- Don't just defend - ATTACK
- We have depth - be aggressive
- Force turnovers, create shorthanded chances
- Rotate players to stay fresh

#### PK System - Passive Box (PROTECT LEAD)

**When to Use**: Late in game protecting lead, 5v3 situations

**Formation**:
```
    F2 -------- F3
     |          |
     |          |
    D1 -------- D2
```

**Keys**:
- Compact box (12 feet)
- Protect slot
- Block shots
- Safe clears only

## Line Deployment Strategies

### Offensive Zone Faceoffs

**Best Offensive Players**:
1. **Line 1** (elite tier forwards)
2. **Line 2** (skilled tier forwards)

**Strategy**:
- Use best skill players
- Quick strike mentality
- Capitalize on opportunity

### Defensive Zone Faceoffs

**Best Defensive Players**:
1. **Strong two-way forwards** (1.50 F/D players and 1.75 F)
2. **Top D pairs** (1.75 D with 2.00 D or F/D partner)

**Strategy**:
- Win draw, clear zone
- Protect lead
- Simple, safe play

### Neutral Zone Faceoffs

**Flexible**:
- Match opponent strength
- Can use any line
- Fresh legs matter

### Late Game Situations

**Protecting Lead (Last 2 Minutes)**:
- Line 1: Elite line (if fresh)
- Line 2: Skilled line (if Line 1 tired)
- Defense: Top pair (1.75 D + strong F/D partner)
- **Do NOT use**: Development line or development D pair

**Need Goal (Last 2 Minutes)**:
- Line 1: Elite line
- Pair 1: Top D pair or elite F/D shifted to D
- Pull goalie at 90 seconds if down 1, 2 minutes if down 2
- **Extra Attacker**: 1.50 skilled forward (responsible, two-way)

### Overtime Strategy (3v3 or 4v4)

**3-on-3 OT Personnel**:
- **Shift 1**: Two 1.00 elite F/D + 1.75 D (elite shift)
- **Shift 2**: 1.50 F/D + 1.50 F/D + 2.00 D (two-way shift)
- **Shift 3**: 1.50 F + 1.75 F + 2.00 F/D (balanced shift)
- Rotate short shifts (45-60 seconds max)

**4-on-4 OT Personnel**:
- **Line 1**: Two 1.00 elite F/D / 1.75 D + 2.00 F/D
- **Line 2**: Two 1.50 F/D / 2.00 D + 2.00 F/D
- Standard shifts (60-90 seconds)

## Game Situation Adjustments

### When Losing (By 1-2 Goals)

**First Period**:
- Stay calm, stick to system
- Don't abandon four-line approach yet
- Assess what's not working

**Second Period**:
- Increase Line 1 ice time by 5%
- Decrease Line 4 ice time by 5%
- Look for momentum shifts
- Stay disciplined (don't take penalties)

**Third Period**:
- Go to three-line system if needed
- Double-shift top line
- Aggressive offensive zone pressure
- Pull goalie at appropriate time

### When Winning (By 1-2 Goals)

**First Period**:
- Keep foot on gas
- Don't sit back early
- Four-line depth maintains energy

**Second Period**:
- Maintain structure
- Continue four-line approach
- Stay aggressive (but smart)

**Third Period**:
- Smart defensive play
- Roll four lines (fresh legs protect lead)
- Block shots, clear zones
- Don't turtle - controlled aggression
- Trust our depth advantage

### When Tied (Close Game)

**Throughout Game**:
- Four-line approach (trust our depth)
- Grind them down
- Stay disciplined
- Win battles
- Our depth emerges over time

**Late Third Period**:
- Slightly increase top line time
- Defensive responsibility increases
- Take smart shots (rebounds count)
- Manage clock awareness

### When Dominating (Winning By 3+)

**Throughout Game**:
- Roll four lines evenly
- Give development forward extra ice time
- Development D gets safer minutes
- Development focus
- Prepare backup goalie to play

## Player Rotation Guidelines

### Forward Shift Length

**Line 1**: 60-90 seconds (can extend in critical situations)
**Line 2**: 60-90 seconds (workhorses, eat minutes)
**Line 3**: 60-75 seconds (high energy, shorter shifts)
**Line 4**: 45-60 seconds (keep development F fresh, limit exposure)

### Defense Shift Length

**Pair 1**: 60-90 seconds (can extend if needed)
**Pair 2**: 60-90 seconds (balanced, reliable)
**Pair 3**: 45-60 seconds (protect development D, shorter shifts)

### Line Matching Philosophy

**Our Approach**:
- Don't obsess over matchups (we have depth)
- Trust all four lines (except Line 4 in critical situations)
- Use last change at home to get top line on their weak line
- On road, trust the system

**Against Elite Opponents**:
- Line 1 vs Their Line 1 (match up)
- Lines 2-3-4 vs Their Lines 2-3 (we have advantage)

## Special Deployment: F/D Versatile Players

### Six Versatile Players (see `roster.md` for current names)

1. **Two 1.00-rated F/D** - Elite at both F and D
2. **One 1.50-rated F/D** - Strong at both F and D
3. **One 1.50-rated F/D** - Primarily F, can play D
4. **Two 2.00-rated F/D** - Solid at both F and D

### When to Move Players

**Move Forward to Defense**:
- Defense injury/absence during game
- Need more defensive structure
- Protecting a lead
- Opponent has weak offensive attack

**Move Defense to Forward**:
- Forward injury/absence during game
- Need more offensive firepower
- Chasing a goal
- Have too many defensemen (rare)

### Common Adjustments

**Most Flexible**: 1.00-rated elite F/D players (elite at both)
**Easy Moves**: 1.50 and 2.00-rated F/D players (good at both)
**Emergency Only**: F/D players who are primarily forward but can help on D

**Example Mid-Game Adjustment**:
- If down a forward: Move a 2.00 F/D to forward, balance defensive pairs
- If down a defenseman: Move a 1.50 F/D to defense, balance forward lines

## Goalie Rotation Strategy

### Season-Long Approach

**Philosophy**: Both goalies split time roughly 50/50

**Goalie 1**:
- Solid all-around
- Can play forward in emergency
- Give equal opportunities

**Goalie 2**:
- Experienced, can play out
- Can play defense in emergency
- Give equal opportunities

### Game Assignment Factors

**Start Goalie 1 When**:
- Against skill-based, high-shot teams (rely on positioning)
- Back-to-back games (alternate)
- Strong recent performance

**Start Goalie 2 When**:
- Against physical, grinding teams
- Back-to-back games (alternate)
- Strong recent performance

**Key Principle**: Recent performance matters most. Ride the hot goalie, but don't completely abandon the other.

### Tournament Strategy

**Pool Play**: Split starts (both get experience)
**Playoffs**: Hot goalie gets preference
**Back-to-Back Days**: Split duty (fresh goalie each game)

## Keys to Success

### Tactical Discipline

1. **Stay in system** - Don't freelance
2. **Support each other** - Five players working together beats five individuals
3. **Backchecking is mandatory** - Defense starts with forwards
4. **Simple play wins** - Don't overcomplicate

### Line Chemistry Factors

**Good Line Chemistry**:
- Complementary speeds (one fast, one smart)
- Communication on ice
- Willingness to work for each other
- Understanding of roles

**Bad Line Chemistry**:
- All three players want the puck
- Speed mismatches (one too fast/slow)
- Lack of communication
- Unclear roles

### Adjusting on the Fly

**Trust Your Depth**:
- If a line is struggling, try different combinations
- Use F/D players to solve problems
- Don't panic - we have options

**Communication**:
- Tell players why you're making changes
- Explain adjustments clearly
- Keep everyone engaged
