# kiro-spec-driven-development Installation

## Quick Install

1. Download: `kiro-spec-driven-development-v1.0.0.zip`
2. Extract to your Claude Code skills directory:
   ```bash
   # Extract to .claude/skills/
   unzip kiro-spec-driven-development-v1.0.0.zip -d ~/.claude/skills/

   # Or for a specific project
   unzip kiro-spec-driven-development-v1.0.0.zip -d /path/to/project/.claude/skills/
   ```
3. The skill will be automatically available in Claude Code

## Verification

Check that the skill was installed correctly:
```bash
ls ~/.claude/skills/kiro-spec-driven-development/SKILL.md
```
