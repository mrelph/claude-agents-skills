# tax-preparation Installation

## Quick Install

1. Download: `tax-preparation-v1.3.0.zip`
2. Extract to your Claude Code skills directory:
   ```bash
   # Extract to .claude/skills/
   unzip tax-preparation-v1.3.0.zip -d ~/.claude/skills/

   # Or for a specific project
   unzip tax-preparation-v1.3.0.zip -d /path/to/project/.claude/skills/
   ```
3. The skill will be automatically available in Claude Code

## Verification

Check that the skill was installed correctly:
```bash
ls ~/.claude/skills/tax-preparation/SKILL.md
```
