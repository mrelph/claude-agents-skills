# research-consolidator Installation

## Quick Install

1. Download: `research-consolidator-v1.0.0.zip`
2. Extract to your Claude Code skills directory:
   ```bash
   # Extract to .claude/skills/
   unzip research-consolidator-v1.0.0.zip -d ~/.claude/skills/

   # Or for a specific project
   unzip research-consolidator-v1.0.0.zip -d /path/to/project/.claude/skills/
   ```
3. The skill will be automatically available in Claude Code

## Verification

Check that the skill was installed correctly:
```bash
ls ~/.claude/skills/research-consolidator/SKILL.md
```
