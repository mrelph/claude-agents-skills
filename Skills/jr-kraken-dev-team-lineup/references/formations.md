# Formations & Tactical Systems - Jr. Kraken Dev Team

## Overview

This document outlines tactical formations, line deployment strategies, and systematic approaches for Coach Mark's Jr. Kraken Development Team. All personnel references use tier/rating designations -- see `roster.md` for current player names and ratings.

---

## Forward Line Structures

### Four-Line System (PRIMARY APPROACH)

**When to Use**:
- Most regular season games
- Pool play in tournaments
- Games where we want to wear opponents down
- When we have full roster available
- Against teams that use 2-3 lines (exploit depth advantage)

**Philosophy**: Roll four purposeful lines, keep everyone fresh, dominate third period

**Line Structure**:

```
LINE 1 - Offensive (35-40% ice time)
Top-ranked forwards (elite tier, 1.00-1.50)
Focus: Offensive production, top matchup line, all special teams

LINE 2 - Two-Way (30-35% ice time)
Strong-tier forwards (1.50-1.75)
Focus: Two-way play, reliable in all situations, PP2 and PK

LINE 3 - Energy/Compete (20-25% ice time)
Solid contributors (1.75-2.00)
Focus: Energy, physicality, forecheck, defensive responsibility

LINE 4 - Depth/Matchup (10-15% ice time)
Remaining contributors with defined roles
Focus: Specific assignments, energy shifts, compete level
```

**Rotation Strategy**:
- Line 1 → Line 2 → Line 3 → Line 4 → repeat
- Shorten Line 4 shifts (45-60 seconds vs 60-90 for Lines 1-3)
- Double-shift Line 1 in critical situations
- Line 3 can play more if game demands additional energy

**Advantages**:
- Every line has purpose and identity
- Fresh legs all game
- Third period dominance (opponents tire, we don't)
- Can handle absences easily
- All players contribute meaningfully

### Three-Line System (COMPETITIVE GAMES)

**When to Use**:
- Close playoff games
- Against very strong opponents
- Late in close games
- When short-handed due to absences
- When we need maximum firepower

**Philosophy**: Best players get more ice time, every player still contributes

**Line Structure**:

```
LINE 1 - Offensive (40-45% ice time)
Top-ranked forwards (elite tier)
Focus: Carry the offense, dominate shifts

LINE 2 - Two-Way (35-40% ice time)
Strong two-way forwards
Focus: Two-way dominance, eat minutes

LINE 3 - Energy/Depth (20-25% ice time)
Solid contributors
Focus: High-energy shifts, simple play, tire opponents

SPOT DUTY:
- Remaining forwards: 5-8 minutes, specific situations
```

**Rotation Strategy**:
- Line 1 → Line 2 → Line 1 → Line 3 → repeat (double-shift top line)
- Spot duty players fill in on changes
- Line 1 and 2 can play back-to-back shifts if needed
- Line 3 matches against opponent's weaker lines

## Defense Pair Structures

### Three-Pair System (STANDARD)

**When to Use**: Most games with full roster

**Pair Structure**:

```
PAIR 1 - Shutdown (40-45% ice time)
Best D-man + strong F/D versatile player
Focus: Against opponent's top line, defensive responsibility, PK

PAIR 2 - Two-Way/Balanced (35-40% ice time)
Next-best D-man + F/D player
Focus: All situations, reliable, can jump into rush

PAIR 3 - Depth (20-25% ice time)
Remaining D-capable players paired by complementary strengths
Focus: Defined role, play to strengths, compete level
```

**Note**: Actual pairs vary based on who's playing forward vs defense

**Rotation Strategy**:
- Pair 1 → Pair 2 → Pair 3 → repeat
- Shorten Pair 3 shifts if needed (45-60 seconds)
- Can run Pairs 1 and 2 back-to-back if needed

**Advantages**:
- Fresh legs on defense all game
- Flexibility to move players forward if needed
- Strong PK depth

### Four-Pair System (TOURNAMENT DEPTH)

**When to Use**:
- Multi-game weekends
- Pool play in tournaments
- When everyone is available
- To maximize ice time distribution

**Pair Structure**:

```
PAIR 1: Top D-man + elite F/D player
PAIR 2: Strong D-man + strong F/D player
PAIR 3: Solid D players paired together
PAIR 4: Remaining D-capable players
```

**Rotation**: Equal-ish shifts (Pair 4 slightly less)

**Advantages**:
- Maximum freshness
- Everyone contributes
- Can handle multiple games
- Full deployment

### Two-Pair System (EMERGENCY/COMPETITIVE)

**When to Use**:
- Multiple D absences
- Very close, critical games
- Overtime situations
- Late in close games

**Pair Structure**:

```
PAIR 1: Best D-man + elite F/D player
PAIR 2: Next-best D-man + strong F/D player
```

**Rotation**: Back-and-forth, 60-90 second shifts

## Special Teams Formations

### Power Play Systems

#### PP1 - Umbrella Formation (PRIMARY)

**When to Use**: Standard power play, against box or diamond PK

**Formation**:
```
         Net-front F (high-ranked)
              |
              |
    Left F --- Quarterback F --- Right F
    (ranked)   (top ranked)     (ranked)

         Point D (strong D)
```

**Roles**:
- **Quarterback (C)**: Setup man, high slot, pass options
- **Left F**: Half-wall, shot/pass threat
- **Right F**: Half-wall, shoot/pass
- **Net-front F**: Screens, tips, rebounds
- **Point D**: Blue line, one-timer threat, pass option

**Keys**:
- Puck movement, tire PK
- Get net-front player in position for screens
- Shoot from point for tips/rebounds
- Half-wall players: shoot or pass threats

#### PP2 - Overload Formation (SECOND UNIT)

**When to Use**: Second power play unit, change of pace

**Formation**:
```
         Net-front F
              |
              |
    Left F - Center F - Right F

         Point D
```

**Philosophy**: Shoot first, crash net, get pucks on goal

**Keys**:
- Simple, direct
- Shots from everywhere
- Crash for rebounds
- High energy

### Penalty Kill Systems

#### PK System - Box+1 Aggressive (PRIMARY)

**When to Use**: Most penalty kill situations

**Formation (5v4)**:
```
        Forward (F1)
            ↑
            | (pressure)

    F2 ---- D1 ---- D2
            |
           F3/F4
```

**Standard PK1 Personnel**: Best defensive-minded forwards + top D-pair

**Keys**:
- F1 pressures puck carrier aggressively
- Box stays compact (15 feet)
- Rotate when F1 tired
- Block passing lanes
- Clear pucks hard when possible

**Philosophy**:
- Don't just defend -- ATTACK
- We have depth -- be aggressive
- Force turnovers, create shorthanded chances
- Rotate players to stay fresh

#### PK System - Passive Box (PROTECT LEAD)

**When to Use**: Late in game protecting lead, 5v3 situations

**Formation**:
```
    F2 -------- F3
     |          |
     |          |
    D1 -------- D2
```

**Keys**:
- Compact box (12 feet)
- Protect slot
- Block shots
- Safe clears only

## Line Deployment Strategies

### Offensive Zone Faceoffs

**Best Offensive Players**:
1. Line 1 (top-ranked forwards)
2. Line 2 (strong two-way forwards)

**Strategy**:
- Use best skill players
- Quick strike mentality
- Capitalize on opportunity

### Defensive Zone Faceoffs

**Best Defensive Players**:
1. Top defensive-minded forwards from Lines 1-2
2. Top shutdown defense pairs

**Strategy**:
- Win draw, clear zone
- Protect lead
- Simple, safe play

### Neutral Zone Faceoffs

**Flexible**:
- Match opponent strength
- Can use any line
- Fresh legs matter

### Late Game Situations

**Protecting Lead (Last 2 Minutes)**:
- Line 1 (if fresh)
- Line 2 (if Line 1 tired)
- Top defense pairs only
- **Deploy top-ranked players to close it out**

**Need Goal (Last 2 Minutes)**:
- Line 1 with top D-pair
- Pull goalie at 90 seconds if down 1, 2 minutes if down 2
- **Extra Attacker**: Best skilled forward from Line 2

### Overtime Strategy (3v3 or 4v4)

**3-on-3 OT Personnel**:
- **Shift 1**: Two best forwards + top D-man (elite shift)
- **Shift 2**: Next two best forwards + second D-man (strong shift)
- **Shift 3**: Next skilled forwards + third D-man (balanced shift)
- Rotate short shifts (45-60 seconds max)

**4-on-4 OT Personnel**:
- **Line 1**: Two best forwards / top D-pair
- **Line 2**: Next two best forwards / second D-pair
- Standard shifts (60-90 seconds)

## Game Situation Adjustments

### When Losing (By 1-2 Goals)

**First Period**:
- Stay disciplined, trust the systems
- Don't abandon four-line approach yet
- Assess what's not working

**Second Period**:
- Increase Line 1 ice time by 5%
- Decrease Line 4 ice time by 5%
- Look for momentum shifts
- Stay disciplined (don't take penalties)

**Third Period**:
- Go to three-line system if needed
- Double-shift top line
- Aggressive offensive zone pressure
- Pull goalie at appropriate time

### When Winning (By 1-2 Goals)

**First Period**:
- Keep foot on gas
- Don't sit back early
- Four-line depth maintains energy

**Second Period**:
- Maintain structure
- Continue four-line approach
- Stay aggressive (but smart)

**Third Period**:
- Smart, structured play
- Roll four lines (fresh legs protect lead)
- Block shots, clear zones
- Don't turtle -- controlled aggression
- Trust our depth advantage

### When Tied (Close Game)

**Throughout Game**:
- Four-line approach (trust our depth)
- Compete hard every shift
- Stay disciplined
- Win battles
- Our depth emerges over time

**Late Third Period**:
- Slightly increase top line time
- Defensive responsibility increases
- Take smart shots (rebounds count)
- Manage clock awareness

### When Dominating (Winning By 3+)

**Throughout Game**:
- Roll four lines evenly
- Get all players meaningful action
- Work on systems and execution
- Prepare backup goalie to play

## Player Rotation Guidelines

### Forward Shift Length

**Line 1**: 60-90 seconds (can extend in critical situations)
**Line 2**: 60-90 seconds (workhorses, eat minutes)
**Line 3**: 60-75 seconds (high energy, shorter shifts)
**Line 4**: 45-60 seconds (keep it tight, defined purpose)

### Defense Shift Length

**Pair 1**: 60-90 seconds (can extend if needed)
**Pair 2**: 60-90 seconds (balanced, reliable)
**Pair 3**: 45-75 seconds (play to their strengths, manage deployment)

### Line Matching Philosophy

**Our Approach**:
- Don't obsess over matchups (we have depth)
- Trust all four lines within their defined purpose
- Use last change at home to get top line vs their weak line
- On road, trust the system

**Against Elite Opponents**:
- Line 1 vs Their Line 1 (match up)
- Lines 2-3-4 vs Their Lines 2-3 (we win here with depth)

## Special Deployment: F/D Versatile Players

### Versatile Players (see `roster.md` for current list)

F/D versatile players are this team's strategic Swiss Army knife. They can play both forward and defense effectively.

### When to Move Players

**Move Forward to Defense**:
- Defense injury/absence during game
- Need more defensive structure
- Protecting a lead
- Opponent has weak offensive attack

**Move Defense to Forward**:
- Forward injury/absence during game
- Need more offensive firepower
- Chasing a goal
- Have too many defensemen (rare)

### Common Adjustments

**Most Flexible**: Elite-rated F/D players (move them freely)
**Easy Moves**: Strong-rated F/D players (good at both)
**Emergency Only**: Players who primarily play one position

**Example Mid-Game Adjustment**:
- If down a forward: Move a depth F/D player to forward, balance defensive pairs
- If down a defenseman: Move a strong F/D player to defense, balance forward lines

## Goalie Rotation Strategy

### Season-Long Approach

**Philosophy**: Goalie starts based on performance, opponent strength, and game situation

### Game Assignment Factors

**Start when**:
- Strong recent performance
- Favorable matchup
- Well-rested (check back-to-back situations)
- Communicated 24-48 hours ahead

**Key Principle**: Recent performance matters most. Ride the hot goalie, but keep both goalies engaged and prepared.

### Tournament Strategy

**Pool Play**: Both goalies get starts (both get experience)
**Playoffs**: Hot goalie gets preference
**Back-to-Back Days**: Split duty (fresh goalie each game)

## Keys to Success

### Tactical Discipline

1. **Stay in system** -- Don't freelance
2. **Support each other** -- Five players working together beats five individuals
3. **Backchecking is mandatory** -- Defense starts with forwards
4. **Execute with purpose** -- Every shift, every play has intent

### Line Chemistry Factors

**Good Line Chemistry**:
- Complementary speeds (one fast, one smart)
- Communication on ice
- Willingness to compete for each other
- Understanding of roles and purpose

**Bad Line Chemistry**:
- All three players want the puck
- Speed mismatches (one too fast/slow)
- Lack of communication
- Unclear roles

### Adjusting on the Fly

**Trust Your Depth**:
- If a line is struggling, try different combinations
- Use F/D players to solve problems
- Don't panic -- we have options

**Communication**:
- Communicate change rationale to players
- Explain adjustments clearly
- Keep everyone engaged and competing
